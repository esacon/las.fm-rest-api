Artist Information API

Welcome to the Artist Information API, where you can retrieve information about artists.

Endpoint

To retrieve information about an artist, make a GET request to the following endpoint:

http://localhost:3000/artist

Query parameters

The API supports two query parameters:

  • name: The name of the artist you want to retrieve information about. If no name is provided, a random artist will be selected.
  • filename: The name of the file where the result will be saved in CSV format. If no filename is provided, a default name will be generated.

Example

Here is an example URL that searches for the artist "Coldplay" and saves the result to a file named "artists":

http://localhost:3000/artist?name=coldplay&filename=artists

Response

The API will return a JSON object with the following information about each artist that coincided with search params:

  • name: The name of the artist.
  • mbid: The MusicBrainz Identifier of the artist.
  • url: The URL of the artist on last.fm.
  • image_small: A small image of the artist.
  • image: A list of images of different sizes for the artist.

The response will be also save in the specified CSV file.
